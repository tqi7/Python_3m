١.انسخ كود بايثون التالي بالضغط على علامة المربع في زاوية الكود (النسخ).

```
list=[['Mecca','Medina'],[100,200],['Riyadh','Jeddah','Dammam']]
#اطبع العنصر الأول#
print(list[0])
print(list[0][0])
print(list[0][1])
#اطبع العنصر الثاني#
print(list[1])
print(list[1][0])
print(list[1][1])
#اطبع العنصر الثالث#
print(list[2])
print(list[2][0])
print(list[2][1])
print(list[2][2])
```
